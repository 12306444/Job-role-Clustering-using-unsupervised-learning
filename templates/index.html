How It Works

The AI analyzes your job description through an advanced NLP pipeline:

  • Text Processing: Cleans and tokenizes the input text
  • Embedding Generation: Converts text to vector using Sentence-BERT
  • Similarity Matching: Compares with known role clusters using cosine similarity
  • Confidence Calculation: Determines prediction confidence based on similarity distance

How It Works

Our unsupervised ML pipeline processes job descriptions without pre-labeled training data.

Text Input

Job description text is cleaned and prepared for analysis

Sentence Embeddings

Sentence-BERT converts text into 768-dimensional vectors

Clustering

HDBSCAN identifies natural role clusters in the embedding space

Similarity

Cosine similarity finds the closest matching role cluster

Prediction

Role is predicted with a confidence score based on distance
